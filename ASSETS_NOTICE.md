## Asset Source

The assets were extracted from the **pokeyellow disassembly**, a community-led
reverse-engineering project:

- **Repository:** https://github.com/pret/pokeyellow
- **Organization:** [pret](https://github.com/pret) (Pokémon Reverse Engineering Tools)
- **Community:** [Discord — pret #pokered](https://discord.gg/d5dubZ3)

The pret disassembly project is a volunteer effort that documents the original
game's source code and data. No original source code from Nintendo or Game Freak
was used — the disassembly was produced entirely through clean-room reverse
engineering by the pret community.
